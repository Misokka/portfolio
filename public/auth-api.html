API d'authentification Ionic

Description

Cette API d'authentification Ionic, réalisé lors de mon BTS à proméo Compiègne en octobre 2022, est une API qui permet de s'authentifier et de récuperer des données d'une application mobile. Pour le projet en question, cette API a été reliée à une application codée grace à Ionic qui se trouve être une liste de course.
